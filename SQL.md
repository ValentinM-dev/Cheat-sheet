## SQL 
### Commande utile
- Extraire une data de la database : `SELECT` | Extraire une data une seule fois de la database : `SELECT DISTINCT` | Extraire d'une data sans doublon avec le nombre de doublon : `SELECT COUNT (DISTINCT ...)`
- Mettre à jour une data de la database : `UPDATE`
- Supprimer une data de la database : `DELETE`
- Ajouter une data dans la database : `INSERT INTO`
- Crée une database : `CREATE DATABASE`
- Modifier une database : `ALTER DATABASE`
- Crée une table : `CREATE TABLE`
- Modifiée une table : `ALTER TABLE`
- Supprimer une table : `DROP TABLE`
- Crée un index : `CREATE INDEX`
- Supprimer un index : `DROP INDEX`
- Indiquer une condition `WHERE`
